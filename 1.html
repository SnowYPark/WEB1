Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. web brosers recieve HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the bulding blocks of HTML pages. with THML constructs, images and other objects, such as interactive forms, may be embedded into semantics for text such as hedings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, writtem using angle brackets.
